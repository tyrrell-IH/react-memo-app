## メモアプリ起動までの手順

### 1.インストール

任意のディレクトリ上で

```
git clone -b dev-Context https://github.com/tyrrell-IH/react-memo-app.git
```

を実行してください。

### 2.ディレクトリの変更

ディレクトリを`react-memo-app/my-memo`へ変更してください。

```
cd ./react-memo-app
```

### 3.npm install

```
npm install
```

を実行してください。

## 起動

```
npm start
```

でメモアプリが起動します。
